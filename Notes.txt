Form 
  Name - Name to the form (user defined)
  method - post / get
  action= link to url / file when click the submit button
  target=_balnk (new window /tab)/ _self (same window) /_parent (parent frame) / frameName
  autocomplete=on/off (enable auto complete popups)
  novalidate - to disable validation 

  

Input
  type (attribute)=  button/reset/submit/checkbox/color/date/datetime-local/email/file/hidden/image/month/number/password/radio/range/search/tel/text/time/url/week 
  Value = The text which displays on the control
  Name = name for the control
  id=name for the control
  width
  height
  autofocus
  hidden
  readonly
  placeholder - to show the hint (supports : text, search, url, tel, email, and password)
  pattern - to set input pattern (supports : text, date, search, url, tel, email, and password) Ex: pattern="[A-Za-z]{3}" title="Country Code must be 3 letters"> / pattern="[0-9]{4}-[0-9]{4}-[0-9]{4}-[0-9]{4} title="Aadhaar Number""
  required - to set mandatory fields (Supports: text, search, url, tel, email, password, date pickers, number, checkbox, radio,file)
  step- to set intervals (increment/decrement) (Supports : number, range, date, datetime-local, month, time and week)  Ex: <input type="number" id="fontSize" name="FontSize" step="5">
  formaction="url" -- to override the Form action attribute (Supports Submit / image)
